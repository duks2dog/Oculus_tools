# Oculus_tools
 社内テックキャンプグループに参加している中で、2020年度作業したOculusに関する実装のうち<br>
 公開可能な一部のソースを格納する。<br>
 開発内容：<br>
 360度カメラ(THETA V)を使用した動体検知とOculusでのVR空間を組み合わせて、<br>
 社内の状態とVR空間を同期できないかの試み。<br>
 
# 仕組み
## カメラの動体検知(WebCamController.cs)
 * OpenCVを使用して、ライブストリーミング映像をIMPUTとした映像を2値化している。
 * グレースケール化、2値化、白黒の反転の順で変換している。

## ライブ映像のSkybox反映(WebCamDrawer.cs)
 * 動体検知以外に、VR上にカメラのライブ映像を取り込んで直接人の状況を見ることができるようにする。
 * THETA Vを検知、INPUTとしてSkyboxへ反映を目的としている。
